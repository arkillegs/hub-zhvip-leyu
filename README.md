# hub-zhvip-leyu

本仓库用于归档与发布多个独立 HTML 页面，内容涵盖不同用途的静态网页资源。仓库主要作为页面存档与展示使用，不针对任何特定域名或网站。

## 项目简介

本仓库收集并整理了一系列 HTML 文件，每个文件均为独立完整的页面。这些页面可能包含不同的布局、样式或功能示例，适合作为静态资源参考、学习或存档用途。

## 目录说明

```
.
├── index.html          # 仓库入口页面，汇总所有归档页面的链接
├── pages/              # 存放各个独立 HTML 页面的文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 页面所需的静态资源（可选）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库首页，提供对所有归档页面的导航。
- `pages/`：核心目录，每个 HTML 文件代表一个独立的页面。
- `assets/`：存放页面共用的样式、脚本或图片资源（如存在）。

## 页面归档说明

所有归档页面均为独立 HTML 文件，可直接在浏览器中打开运行。页面之间无相互依赖，便于单独查看与使用。

归档页面可能包含以下类型：
- 静态内容展示页
- 功能或交互示例页
- 布局或设计参考页

## 维护说明

- 本仓库定期更新，新增或更新归档页面。
- 提交新页面时，请将 HTML 文件放入 `pages/` 文件夹，并在 `index.html` 中添加对应链接。
- 如页面需要额外资源，请统一放入 `assets/` 对应子目录中，避免文件分散。
- 请勿在页面中嵌入任何指向未知或不可信外部资源的链接，保持页面自包含。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `index.html` 查看归档页面列表。
3. 点击链接或直接打开 `pages/` 下的任意 HTML 文件即可浏览对应页面。
